Учебное веб-приложение для бронирования вилл с использованием ASP.NET Core и принципов Clean Architecture.

Реализована функциональность просмотра номеров, оформления бронирования и управления заказами.

Использовал Entity Framework Core (Code First, миграции), ASP.NET Core Identity для аутентификации и авторизации, а также применил Repository Pattern.

Дополнительно реализована административная панель с управлением бронированиями, интеграция платежей (Stripe), а также работа с отчетами и графиками.
